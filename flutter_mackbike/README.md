# MackBike

A bike navigation app created with Flutter.

## Features

*   **Interactive Map:** Displays an interactive map using OpenStreetMap.
*   **"My Location":** A button to center the map on the user's current location.
*   **Search:** Search for origin and destination addresses with autocomplete.
*   **Routing:** Generates a bike-friendly route between the origin and destination using a custom API, displaying the route on the map along with distance and turn-by-turn instructions.
*   **Turn-by-Turn Navigation:** Provides live, turn-by-turn navigation with automatic instruction progression and real-time location tracking on the map.
